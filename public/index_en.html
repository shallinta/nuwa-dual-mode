One Mental Model, Two Faces

🧠

Distill Cognitive Operating Systems

Not surface imitation. Extract how they think, decide, speak, and what they won't do. Mental models split into two columns: Value Lenses + Operational Techniques.

🎯

Auto-Infer Intents — No Asking

女娲 infers all possible intent dimensions from the subject's profile — methodology learning, style imitation, roleplay, decision review… Never asks "what do you want this for?"

🔒

Care Red Line — Hard Gate

When emotional companionship probability > 0, the artifact must include crisis hotlines for CN / EN / SG / HK / TW. Persona will break character temporarily to provide real help resources.

⚡

6 Agents in Parallel

Writings, conversations, expression DNA, external perspectives, decisions, timeline — six dimensions researched in parallel. Five minutes covers two hours of solo research.

🔄

Incremental Updates

Already-distilled skills support incremental updates — only pull the latest. Three choices: overwrite / incremental (recommended) / new v2 alongside.

🌐

50+ Runtime Universal

Built on the open Agent Skills protocol. Runs in Claude Code, Codex, Cursor, OpenClaw, Hermes, Gemini CLI and more.

Distillation Pipeline

From input to delivery — 9 steps auto-flow. Red markers are explicit checkpoints requiring user confirmation.

🔍
Phase 0 · Checkpoint ✓
Entry Routing + Existing Skill Detection
Identify the subject's profile, region, and primary language. Scan locally for existing skills with the same name — if found, trigger three-way choice: overwrite / incremental update / new v2 alongside.
Profile ScanLanguage DetectionUpdate Check
🎯
Phase 0A · Checkpoint ✓
Auto Intent Inference + Form Selection
Infer 12+ intent dimensions from subject profile (methodology / style imitation / roleplay / emotional companionship / decision review…). Extrapolate unique dimensions. Auto-determine artifact form (dual/single mode) — present to user for confirmation. First-time terms explained in plain language.
12+ IntentsAuto-DetectTerm Explanation
⚡
Phase 1
6-Agent Parallel Research
Writings / conversations / expression DNA / external views / decisions / timeline — six dimensions run simultaneously. Source whitelist auto-adjusted by subject's language and region. Primary sources first; Zhihu/WeChat/Baidu Baike permanently excluded.
Parallel ResearchSource FilteringBilingual Strategy
📊
Phase 1.5 · Checkpoint ✓
Research Quality Summary
Generate primary/secondary source ratio, cross-agent contradiction detection, under-informed dimension flags. Under 10 total sources → recommend lowering expectations or supplementing research.
Quality AuditContradictionsSource Ratio
🧠
Phase 2
Mental Model Framework Extraction
Triple Validation extraction of core mental models. Two columns: Part A — Value Lenses (2-5) — cross-domain recurrence / generative power / distinctiveness; Part B — Operational Techniques (1-5) — personally-named actionable checklists. Fan-work vs official conflicts trigger pause for user decision.
Triple ValidationTwo ColumnsFan-Work Negotiation
🏗️
Phase 3
Skill Construction & Injection
Build 3-layer directory (shared / mode-persona / mode-toolkit) from templates. Inject feature matrix, Care Red Line (5-region hotlines + fallback), era anchoring statement, tiered disclaimers, timeliness declaration, Agentic Protocol.
Template FillCare Red LineTimeliness
🔬
Phase 4 · Checkpoint ✓
Quality Validation + Repair Cycle
Select validation suite by inferred intents. Toolkit technique enumeration check (all personally-named techniques must be recalled — miss one = FAIL). Care Red Line three-signal simulation (ordinary low / extreme distress / unhealthy dependency) — all three must PASS. Max 2 repair cycles; if still FAIL at cycle 2, user must confirm whether to deliver.
Technique CheckCare Test2-Cycle Limit
✨
Phase 5
Dual-Agent Refinement
Two independent agents review separately: intent dimension coverage completeness + mandatory section injection completeness. Cross-validation with merged recommendations ensures no omissions.
Cross-ValidationIntent CoverageSection Complete

Delivery
Package & Register + 5-Part Conclusion
Upload and register as custom skill. Main message outputs five parts: artifact form / usage (including mode switching and sample queries) / limitation reminders / ongoing maintenance (modify / update / uninstall) / skill summary with 女娲 guidance. None may be omitted.
